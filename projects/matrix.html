Summary

Moving quickly from specifications to polished screens

Matrix is a startup that connects fans with creators and digital brands through rewards and gamified engagement.

I worked with the CEO to turn product requirements and an existing style guide into high-fidelity screens that were ready for development handoff.

Feature Set One

Events and voting

The first feature set focused on helping users browse events and vote for their favorite creators. I translated specs into low-fidelity concepts, iterated with the team, and then built the final interface system.

The resulting screens covered the event browse experience, creator selection, and the voting-results flow tied to live creator streams.

Feature Set Two

Tokens and NFTs

I also designed flows for buying and selling tokens and collectibles. These included market overviews, order management, creator-specific detail pages, and transaction entry points.

The main challenge was keeping dense financial and collectible data legible on mobile while preserving the energy of a creator-focused platform.

Takeaway

Speed matters when clarity is already handled

This project was a concentrated UI exercise in speed, collaboration, and execution. It sharpened my ability to absorb product requirements quickly and still produce screens that feel polished, coherent, and handoff-ready.
